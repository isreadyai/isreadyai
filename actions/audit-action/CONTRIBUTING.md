# Contributing

This repository is a **generated build** of the isready.ai AI-readiness audit
action — see [`SYNCED.md`](./SYNCED.md) for exactly which file here maps to
which source in the monorepo. Everything in this tree, including this file, is
**overwritten** on every sync, so a pull request opened directly against this
repo cannot be merged as-is.

**Please contribute upstream:** open issues and pull requests in the
[`isreadyai/isreadyai`](https://github.com/isreadyai/isreadyai) monorepo. Its
[`CONTRIBUTING.md`](https://github.com/isreadyai/isreadyai/blob/main/CONTRIBUTING.md)
covers getting set up, the check suite to run before opening a PR, and the
recipe for adding a new scanner check.

## Reporting a vulnerability

Please do not open a public issue for security problems — see
[`SECURITY.md`](./SECURITY.md) for private reporting channels.
