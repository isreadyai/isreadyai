# Security Policy

This repository is a **generated build** of the isready.ai AI-readiness **fix PR**
action. Its source lives in the
[`isreadyai/isreadyai`](https://github.com/isreadyai/isreadyai) monorepo — please
do not report issues against the generated files here.

## Reporting a vulnerability

**Please do not open a public GitHub issue for security problems.** Report
privately through one of:

- **GitHub Security Advisories** — use the monorepo's
  [**Report a vulnerability**](https://github.com/isreadyai/isreadyai/security/advisories/new)
  button (Security → Advisories). This is the preferred channel.
- **Email** — `dev@smartsquad.io`
  ([Smart Squad S.r.l.](https://smartsquad.io)), subject `isready.ai security`.

Please include enough detail to reproduce: affected component, a proof of
concept, the impact you observed, and any relevant logs.

The full, authoritative security policy — scope, response targets and process —
is maintained in the monorepo:
<https://github.com/isreadyai/isreadyai/blob/main/SECURITY.md>.
